# AssaultCube-Aimbot
This is a program (C++) that does a simple aimbot to target the closest enemy.

# Offsets
localPlayer = 0x57e0a8;
entityArray = 0x58AC04; // next entity +0x04
playerNumber = 0x591fd4; // Bot Only

# Pointers
headPosition = {0x04, 0x08, 0x0c}; // X, Y, Z
Ammo = 0x138;
Name = 0x205;
